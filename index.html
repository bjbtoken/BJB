How to Buy (Base)

  1. Install MetaMask and select the Base network.
  2. Fund your wallet with a little ETH on Base for gas (bridge from Ethereum if needed).
  3. Click “Add BJB to MetaMask” above, or import the token manually using the address.
  4. Swap on your preferred Base DEX (e.g., Uniswap).
